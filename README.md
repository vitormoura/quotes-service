# quotes-service
REST service to manage motivational or corporate quotes

### Libraries

- FLASK RESTFUL: https://flask-restful.readthedocs.io/en/latest/
- Flask-SQLAlchemy: http://flask-sqlalchemy.pocoo.org/2.3/quickstart/

### Instalation

```shell
> pipenv install  
> pipenv shell
> python main.py
```
